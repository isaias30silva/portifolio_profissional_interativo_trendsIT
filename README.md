# 💼 Meu Portfólio Profissional Interativo

Este projeto consiste no desenvolvimento de um **portfólio web estático
e responsivo**, criado como parte do programa TrendsIT 2026, uma realização do Núcleo Softex Campinas e coordenação do Softex Nacional.

O objetivo principal é atuar como um **cartão de visitas digital**,
apresentando de forma clara e profissional a trajetória, habilidades
técnicas e projetos desenvolvidos pelo estudante.

## 🎯 Objetivo do Projeto

Construir um site estático profissional utilizando **HTML5 e CSS3**,
capaz de:

-   Apresentar informações profissionais de forma clara e organizada\
-   Servir como vitrine de projetos\
-   Ser acessado publicamente por recrutadores\
-   Carregar rapidamente e manter consistência visual\
-   Demonstrar domínio dos fundamentos do desenvolvimento web

------------------------------------------------------------------------

## 🚀 Tecnologias Utilizadas

-   HTML5 (semântica e estrutura)
-   CSS3 (estilização e responsividade)
-   Git & GitHub (versionamento)
-   GitHub Pages (deploy)

------------------------------------------------------------------------

## 🛠️ Ferramentas de Desenvolvimento

-   Visual Studio Code (VS Code)
-   DevTools do navegador
-   Lighthouse (auditoria de performance)

------------------------------------------------------------------------

## 📱 Funcionalidades

-   Página única (Landing Page)
-   Seção "Sobre Mim" com biografia e imagem
-   Galeria de projetos com links para repositórios
-   Integração com LinkedIn e GitHub
-   Seção de contato com múltiplos canais
-   Layout responsivo (Mobile First)

------------------------------------------------------------------------

## 📐 Conceitos Aplicados

Durante o desenvolvimento, foram aplicados conceitos fundamentais como:

-   HTML semântico (`header`, `main`, `section`, `article`, `footer`)
-   CSS Flexbox para layout
-   Responsividade com Media Queries
-   Organização de código e separação de responsabilidades
-   Boas práticas de acessibilidade (uso de `alt`, contraste, etc.)
